# 機能性表示食品データの配置場所

消費者庁の届出データ検索から取得したCSVをここに配置してください。

## 取得方法

消費者庁 機能性表示食品届出データベース（または以下のExcel/CSV）を配置。

## 期待するCSV列（消費者庁公式フォーマット）

```csv
届出番号,商品名,届出者名,機能性関与成分名,表示しようとする機能性,...
A1,○○サプリ,△△株式会社,乳酸菌(BB12株),腸内フローラのバランスを整え...
A2,EPA・DHAサプリ,□□製薬,EPA・DHA,中性脂肪を低下させ...
```

必須列:
- `機能性関与成分名`（または `関与成分` / `素材`）
- `表示しようとする機能性`（または `機能性` / `効果`）

## 処理コマンド

```sh
npx tsx scripts/process-health-foods.ts
```

## 処理内容

1. 成分名 → `baseIngredient`（大本の素材）に自動分類
   - 例: 「りんご由来プロシアニジン」→ りんご
   - 例: 「乳酸菌（BB12株）」→ 乳酸菌・ビフィズス菌

2. 機能性テキスト → 各スコア軸に自動付与
   - 「腸内フローラ」→ gut スコア上昇
   - 「睡眠の質」→ rest スコア上昇

3. 同一成分名の届出件数 → `usageCount` として保存
